# 🧠 Brainrot Dating

Um aplicativo de encontros com conteúdo viral (brainrot memes) e personagens celestiais raros!

## 🎮 Funcionalidades

- ✅ Sistema de cards deslizáveis (swipe left/right)
- ✅ Memes brainrot como conteúdo
- ✅ Chat real-time com personagens celestiais
- ✅ Sistema de raridade (Celestial, Epic, Rare, Common)
- ✅ Autenticação JWT
- ✅ Perfis dinâmicos

## 🛠️ Tech Stack

### Backend
- Node.js + Express
- MongoDB
- Socket.io (chat real-time)
- JWT Authentication

### Frontend
- React Native (Expo)
- Redux para state management
- Socket.io client

## 🚀 Como Começar

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
expo start
```

## 📱 Gameplay

1. Swipe em memes brainrot
2. Faça match com personagens celestiais
3. Converse com eles via chat
4. Desbloqueie personagens raros!
